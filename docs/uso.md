# Uso

## Ejecutar el sitio localmente

```bash
mkdocs serve
```

Luego abre:

```text
http://127.0.0.1:8000
```

## Construir el sitio

```bash
mkdocs build
```

## Publicar en GitHub Pages

```bash
mkdocs gh-deploy
```

!!! note "Publicación"
    MkDocs generará el sitio y lo subirá a la rama `gh-pages`.
